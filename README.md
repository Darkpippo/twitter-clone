# Twitter Clone Demo

Full-stack social app built with **React Native (Expo)** + **NestJS**. 

Features:
- User auth (register/login)
- Tweet feed + composer 
- Profile pages w/ stats
- Follow system (real-time sync)
- Likes + comments

## Quickstart 
**Checkout Backend Readme**

**Backend:**

cd backend

npm run start:dev

**Checkout Frontend Readme**

**Frontend**

cd frontend  

npx run start
